# InsertWrap - 72桁で折り返し改行を入れる Thunderbird アドオン
## 概要

Thunderbirdには標準で72桁での折り返しが設定されてはいますが、あくまで表示上の話であり、送信メールには改行が含まれません。このアドオンはメール送信時に改行を挿入し、実際にメール本文を72桁で折り返させます。

- 全角文字は2文字でカウントするなど日本ユーザー向けの挙動。
- 折り返し文字数の変更機能。
- メール送信時に自動改行、手動での改行、両対応。
- URLや引用行は保護されます。

## インストール方法

Releaseから署名済みxpiをダウンロードし、手元のThunderbirdにインストールしてください。（アドオン設定のページで「ファイルからアドオンをインストール」を選ぶ）

## 動作について

- アドオンを導入すると、以後、メール送信時に72桁で折り返しの改行が入るようになります。
  - URLが途中で切られるような折り返し改行は行いません。
  - 引用（行頭に">"がある行）は折り返しません。
- メール編集画面のツールバーにも「折り返し改行」ボタンが増えます。こちらを押すと即座に折り返し改行が入ります。
- 拡張機能ページで設定を開くことができます。メール送信時の自動改行の無効・有効化、折り返しの桁数の変更ができます。

## ソースコードについて
